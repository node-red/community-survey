# Node-RED Modernization Survey Results

This repository is part of the Node-RED Modernization Project and is used to showcase the result of the Node-RED modernization survey of 2025, with them being fully segmentable.

For more information on the Node-RED Modernization Project please see this [Node-RED Forum post](https://discourse.nodered.org/t/node-red-survey-shaping-the-future-of-node-reds-user-experience/98346).

**Note:** Please reach out through the [official Node-RED issue tracker](https://github.com/node-red/node-red/issues) for any inquieries or feedback for now.
